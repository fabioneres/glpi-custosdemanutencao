# Custos de Manutencao

Plugin GLPI 10 para registro de consumo de materiais de manutenção em chamados, com materiais SINAPI, histórico de preços, centros de custo e relatórios gerenciais.

Esta versão entrega a primeira versão operacional ampliada: estrutura, banco, direitos, menus, cadastros principais, importação CSV/XLSX SINAPI, aba de chamados com preço automático, cancelamento lógico, auditoria, histórico de preços, exportações CSV, relatórios gerenciais e seção Sobre.
